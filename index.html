☁️ Enable Cloud Sync

Sync across all your devices using a private GitHub Gist. One-time setup, takes 2 minutes.

Step 1: Go to github.com/settings/tokens/new
Step 2: Note = "DailyOS", Expiration = "No expiration"
Step 3: Scope = check only "gist"
Step 4: Click "Generate token" and copy it
